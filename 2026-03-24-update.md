# wtfoc — Day 1 Recap & Day 2 Plan

**Yesterday we went from zero to a working end-to-end knowledge graph pipeline.** You can now:

- **Ingest a GitHub repo** (issues, PRs, comments, discussions) into a versioned knowledge graph with one command
- **Trace any concept** across sources — ask "what connects X to Y?" and get a ranked, cited path through the graph
- **Semantic search** over ingested content with pluggable embedders (OpenAI, LM Studio, Ollama)
- **Store collections on Filecoin** as CAR bundles with content-addressed provenance (identity, revisions, diffs)
- **Run the demo**: `./wtfoc` ships as a single binary with a real-data GitHub demo script

13 PRs merged, full CI, release v0.0.1 published.

**Today** we're extending ingest to new source types — Discord, Slack, and web pages — so by end of day you'll be able to point wtfoc at a project's GitHub + Discord + docs site and get a unified knowledge graph across all three.
